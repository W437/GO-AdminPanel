# Scripts Directory

This directory contains utility scripts for development, deployment, and maintenance.

## Scripts

### `optimize-performance.sh`
Performance optimization script for the Laravel application.

**Usage:**
```bash
chmod +x scripts/optimize-performance.sh
./scripts/optimize-performance.sh
```

### `script.sh`
General utility script (check contents for specific usage).

**Usage:**
```bash
chmod +x scripts/script.sh
./scripts/script.sh
```

## Adding New Scripts

When adding new scripts to this directory:

1. Make scripts executable: `chmod +x scripts/your-script.sh`
2. Add a description in this README
3. Include usage instructions
4. Document any dependencies or requirements

## Best Practices

- Always test scripts in a development environment first
- Include error handling in scripts
- Document script purpose and usage
- Use descriptive script names
- Make scripts executable but review before running
